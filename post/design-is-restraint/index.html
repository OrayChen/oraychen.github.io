设计的本质是留白

阅读时间 3 分钟 · 修改于 7 天前
设计思考

# 说在前面

或许我们已经厌倦了那些花里胡哨的设计：从众多按钮、蓝紫渐变配色的背景，到纷繁的特效、动画，审美疲劳已经逐渐形成。换个角度想想，也许美化的尽头就是默认，而简洁至上。

# 加法是本能，留白是修行

人类天生喜欢加法。当我们面对一张空白的画布，第一反应往往是“还能加点什么”。当然是更多颜色、更多动效、更多装饰。

但我们要明白：好的设计是做减法。

# 留白是什么？

留白是中国传统艺术作品创作手法，指通过保留空白空间激发观者想象的艺术表现方式。该手法在书画、雕刻等领域均有应用，强调虚实相生的美学原则。

留白，是一种境界。我们观齐白石的虾，便能感受到水的清澈；赏徐悲鸿的马，便能体味到风的速度。南宋马远的《寒江独钓图》里，一只小舟，一个渔翁在垂钓，整幅画中没有一丝水，而让人感到烟波浩渺，满幅皆水。这便是中国画的最高境界——留白。

画如果过满过实，在构图上就失去了灵动与飘逸，显得死气沉沉；而有了留白，便给予观赏者以遐想和发挥的空间，正所谓“此处无物胜有物”。

可以想象，放到现代设计，这就是一种极简风的体现。它让读者先看到内容，接着给了读者浮想联翩的空间。我们可以看到，Memoire走的正是极简风的路。

"Pare down to the essence, but don't remove the poetry." — Leonard Koren

# 我们该怎么做？

在软件设计中，我们可以：

  • 减少选择：每个屏幕只做一件事
  • 减少色彩：2-3 种主色即可，其余用灰度表现层级
  • 减少动效：一个淡入比十个弹跳更有高级感
  • 减少文字：没有人会读你的欢迎页

# 实践中的例子

css 
/* 过度设计 */
.card {
  background: linear-gradient(135deg, #667eea, #764ba2);
  box-shadow: 0 20px 60px rgba(0,0,0,0.3);
  border-radius: 24px;
  transform: scale(1.05);
}

/* 克制设计 */
.card {
  background: #fff;
  border: 1px solid #e5e5e5;
  border-radius: 8px;
}

第二种方案虽然简单，但在合适的上下文中反而更耐看。

# 写在最后

设计不在于我们加了多少东西，而在于我们敢于删掉多少东西。

下次当我们打开设计工具时，试着问自己："我还能删掉什么？"
