你好，Memoire

阅读时间 2 分钟 · 修改于 2026年6月24日
随想生活

这是我的个人博客，名为Memoire。

目前我并没有那么高超的技术，有很多不懂的地方都借助了claude来实现。

我从未想过我会如此热衷于发博客，或许是因为，我向来都迫切地想要一个发泄我所有想法的一方天地吧。现在，看着这里如此广阔的空白，我心生感慨，我要把这里填满，用记忆的洪水。

先来看看Memoire的设计思路吧。

# 为什么是 Memoire？

「Memoire」是法文中的「记忆」。选择这个名字，是因为我相信写作是对抗遗忘最优雅的方式。

在这个信息碎片化的时代，我们每天都在消费海量内容——社交媒体的推文、短视频的轰炸、即时通讯的闪烁。但真正值得我们停下来、静静书写的时刻，有多少被我们真正记录了下来？

Memoire就是这样的一个——属于我自己的——博客。它让我能够停下来，认真记下每一个值得被铭记的时刻。

# 这不是又一个博客

Memoire 的设计理念从一开始就与普通的博客不同：

  • 没有评论区：这里只是一间书房。你可以安静地阅读、思考，然后离开
  • 没有点赞数：好的文字不需要数字来证明
  • 没有推荐算法：每一篇文章都按时间排列

「其实这只是因为我不想实现这些功能。」大葱如是说道。

# 一些计划

在接下来的文章里，我想写写这些（但不限于这些）话题：

  1. 我的日常
  2. 对于设计与工程的一些idea
  3. 读书笔记与思考
  4. 遇见的风景与人
"Memory is the diary that we all carry about with us." — Oscar Wilde
